Full Handshake, +/-6VDC signal
Computer is DTE, Interface is DCE
1: DCD (DTE-In) = Data Carrier Detect
2: RXD (DTE-In) = Received Data
3: TXD (DTE-Out) = Transmitted Data
4: DTR (DTE-Out) = Data Terminal Ready
5: GND
6: DSR (DTE-In) = Data Set Ready
7: RTS (DTE-Out) = Request To Send
8: CTS (DTE-In) = Clear To Send
9: RI (DTE-In) = Ring Indicator
